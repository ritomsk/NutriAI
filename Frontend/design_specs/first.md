# Design Specification: AI Food Ingredient Guide

**Theme:** "Ethereal Clarity" — Organic, trustworthy, and high-tech.
**Visual Style:** Light Mode, Mesh Gradients, Glassmorphism.

---
**Tech Stack:** React(javascript), Tailwind CSS, Framer Motion
## 1. Color Palette

### Background (The Mesh)
The background is not a static color but a fluid mesh gradient on a white base.
* **Base Canvas:** `#FAFAFA` (Off-White)
* **Gradient Blob A (Trust/Fresh):** `#A7F3D0` (Soft Mint)
* **Gradient Blob B (Caution/Warmth):** `#FECACA` (Soft Coral)
* **Gradient Blob C (AI/Tech):** `#E9D5FF` (Pale Lavender)

### UI Elements (Glass)
* **Card Background:** `#FFFFFF` (White) with `0.6` to `0.8` Opacity.
* **Border:** `1px solid rgba(255, 255, 255, 0.4)`

### Semantic Colors (Status Indicators)
* **Safe / Natural:** `#059669` (Emerald 600)
* **Caution / Processed:** `#D97706` (Amber 600)
* **Avoid / Hazardous:** `#DC2626` (Red 600)
* **Text Primary:** `#1F2937` (Gray 800) — *Never use pure black.*
* **Text Secondary:** `#6B7280` (Gray 500)

---

## 2. Typography

**Primary Font (Headings & UI):** `Inter` or `Plus Jakarta Sans`
* *Why:* Geometric, readable, modern. Fits the "AI" aesthetic.
* *Weights:* SemiBold (600) for headers, Medium (500) for buttons.

**Secondary Font (Body Text):** `Merriweather` or `Roboto`
* *Why:* High readability for dense nutrition facts and explanations.
* *Line Height:* `1.6` for optimal reading comfort.

---

## 3. Layout Structure

### A. The Hero Section (Centered)
* **Vibe:** Minimalist Google-search style but elevated.
* **Component:** Large, pill-shaped input container.
    * *Left:* Search Icon (SVG).
    * *Middle:* Input Text ("Paste ingredients or scan label...").
    * *Right:* Camera Icon (Trigger file upload/camera).
* **Animation:** The background mesh blobs should slowly drift/rotate behind this frosted glass bar.

### B. The Results Grid (Masonry or Grid)
* **Container:** A grid of cards appearing *below* the search bar.
* **Card Anatomy:**
    1.  **Header:** Ingredient Name + Safety Badge (Pill shape).
    2.  **Body:** "Plain English" explanation generated by AI.
    3.  **Footer:** Source/Confidence score (small text).

---

## 4. UI Components & CSS Effects

### The "Glass" Effect (CSS Reference)
Apply this to the Search Bar and Ingredient Cards to achieve the "floating" look over the gradient.

```css
.glass-panel {
  background: rgba(255, 255, 255, 0.65);
  backdrop-filter: blur(12px);
  -webkit-backdrop-filter: blur(12px);
  border: 1px solid rgba(255, 255, 255, 0.3);
  box-shadow: 0 4px 30px rgba(0, 0, 0, 0.05);
  border-radius: 16px;
}
